## Core Features:

- Animated Hero Section: Display Siva Sudhamsh Gandikota's name and title with an animated neural network background, creating an immersive introduction.
- Interactive Skills Showcase: Present Siva's skills in a grid of glassmorphism cards with animated proficiency levels that flip on hover, engaging users with interactive data visualization.

- Project AutoTuning.AI Insights Tool: Create the project by combining the tech stack in the user provided request.  AutoTuning.AI tool displays the project details for a AI assisted software tuning program. The insights includes displaying stats (87% accuracy, 1K+ users, <2min diagnostics). The description should read the important pieces in a short, but memorable and clear blurb. Generative AI as a tool, the tool reasons on displaying a certain insight to the user for better clarity and user comprehension. For example: 'Displays important insights such as a model reaching 87% accuracy after 10,000 loops.'.
- Contact AI Assistant: Include a contact form with an animated AI chatbot offering help, providing an interactive, user-friendly interface for visitors to get in touch with Siva.
- Resume download: Provide a call-to-action (CTA) in the Hero section.

## Style Guidelines:

- Primary color: Deep navy/black (#0a0e27) for a dark, AI-themed aesthetic.
- Accent colors: Electric blue (#00d4ff), Purple (#8b5cf6), and Cyan (#06b6d4) to highlight AI elements and provide a futuristic feel.
- Body font: 'Inter', sans-serif, for its modern and neutral look, suitable for body text. Headline font: 'Space Grotesk', sans-serif, is well suited for the title because it has a computerized and techy feel. Note: currently only Google Fonts are supported.
- Use glowing, tech-specific icons for skills and experience sections to enhance the AI theme. Note: Siva mentions skill cards that glow with 'tech-specific colors'.
- Implement glassmorphism cards for content sections to create depth and a modern UI effect.
- Add subtle animations, such as scroll-based triggers and hover effects, to create a dynamic and engaging user experience.
